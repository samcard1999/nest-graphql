# Dev

1.Clone the project

2.Copy ```env.template``` and rename to ```.env```

3.Execute
``` 
yarn install
```
4.Load the image (Docker Desktop)

5.Load Nest backend 
```
yarn start:dev
```
6.Go to site: 
```
localhost:3000/graphql
```
7. Execute __@Mutation__ executeSeed, to fill the database with information.
